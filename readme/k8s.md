### ☸️ Kubernetes Task: Deploy Your Application

You are required to deploy your application using Kubernetes.

---

### Requirements

1. Use a local Kubernetes cluster (e.g., Minikube)

2. Create a Deployment:

   * Use your Docker image
   * Ensure the application runs inside a pod

3. Expose the application:

   * Create a Service
   * Make the application accessible

4. Verify that:

   * Pods are running
   * Service is accessible

---

### Expected Outcome

* Deployment created successfully
* Pod is running
* Application is accessible from browser

---

### Notes

* If the app is not accessible → check service type and ports
* Debugging is part of the task
